Callback Example - SRP

Solution: Wrap $.ajax in a function that requires a callback argument. 

function getUser(onSuccess) {
  $.ajax({
    url: "user.json",
    type: "GET",
    success: onSuccess
  });
}
Create a second function that handles the rendering of the data. 

function onSuccess(data){
    $.each(data, function(key, value) {
    $("#" + key).text(value);
  });
}

getUser(onSuccess); //call getUser passing in the callback

Problem: What if the data is needed by different modules at different times.
